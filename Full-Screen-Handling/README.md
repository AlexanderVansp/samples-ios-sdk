# THEOplayer iOS Reference Apps - THEO Full Screen Handling

The purpose of this app is to demonstrate how to manage [THEOplayer] full screen mode.

## Getting Started

* Obtain THEOplayer SDK. If you don't have a SDK yet, please visit [Get Started with THEOplayer].
* Copy & paste the unzipped **_THEOplayerSDK.framework_** folder to the root of the reference app project. For example:

        cp -a THEOplayerSDK.framework/ Full-Screen-Handling/
